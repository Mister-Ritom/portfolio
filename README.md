# Ritom Ghosh | Developer Portfolio

An immersive, 3D-driven developer portfolio built with **SvelteKit 5**, **Threlte (Three.js)**, and **GSAP**.

## üöÄ Overview

This is the source code for my personal portfolio. It features a high-performance 3D experience centered around an interactive bike model that reacts to user scroll, showcasing a blend of engineering mindset and creative execution.

## ‚ú® Features

- **Immersive 3D Experience**: Interactive 3D bike model powered by [Threlte](https://threlte.xyz/) and [Three.js](https://threejs.org/).
- **Scroll-Linked Animations**: Smooth, performant transitions and 3D rotations driven by [GSAP ScrollTrigger](https://gsap.com/docs/v3/Plugins/ScrollTrigger/) and [Lenis](https://lenis.darkroom.engineering/) for smooth scrolling.
- **Dynamic Project Showcase**: Automatically pulls and displays project data, including GitHub integration.
- **Premium Aesthetics**: Dark-themed, high-contrast UI built with **Tailwind CSS**, focusing on a professional "builder" aesthetic.
- **Responsive Design**: optimized for both desktop and mobile viewing.
- **Contact Integration**: Fully functional contact form to get in touch.

## üõ†Ô∏è Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm / pnpm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Setup environment variables:
   Create a `.env` file in the root directory (refer to `.env.example` if available).

### Development

Start the development server:
```bash
npm run dev
```

### Building

To create a production version:
```bash
npm run build
```

You can preview the production build with `npm run preview`.
